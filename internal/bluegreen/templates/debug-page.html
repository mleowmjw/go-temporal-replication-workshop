Datastar Quick Reference

AttributePurposeExample
data-signalsDeclare reactive statedata-signals='{"count":0}'
data-on:clickEvent listener (colon!)data-on:click="@post('/x')"
data-on:keydownKeyboard eventdata-on:keydown="..."
data-bind="sig"Two-way bindingdata-bind="myInput"
data-bind:sig-nameTwo-way (kebab→camel)data-bind:foo-bar → $fooBar
data-textSet text contentdata-text="$count"
data-showShow/hide elementdata-show="$isOpen"
@get(url)Fetch via GET (SSE)@get('/endpoint')
@post(url)Fetch via POST (SSE)@post('/endpoint')

Key rule: Event attributes use colon separators (data-on:click), NOT hyphens (data-on-click). Hyphens silently fail — no error, no request.
